# flaskproject


OS: MacOS
Python 3.8.5 (venv)
pip 20.1.1


##Instruction:

1. Run `$ . venv/bin/activate`
2. Run `$ export FLASK_APP=hello.py`
3. Run `$ flask run`<br>
(`* Running on http://127.0.0.1:5000/`)
